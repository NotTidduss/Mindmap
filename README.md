# Mindmap
A small tool developed in the Godot engine, used for connecting notes.

Prefix a line with "Todo:" to add it to the side.
Use the sidebar search field to find entries by title (including fuzzy matches), highlight matches, and see a live match count. Press Esc to clear the search.

## Project Structure
- `scenes/` - Godot scene files.
- `scripts/core/` - Domain/data/manager logic.
- `scripts/ui/` - UI root scripts.
- `assets/icons/` - App icon and import metadata.
- `docs/` - Architecture and planning docs.
